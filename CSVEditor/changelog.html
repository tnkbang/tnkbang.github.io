Phiên bản 2.0.0.2 Version 2.0.0.2 バージョン 2.0.0.2
MỚI NHẤT LATEST 最新
  • Thêm chức năng cho phép chèn một hoặc nhiều cột phía trước cột đang chọn.
  • Added functionality to insert one or multiple columns before the selected column.
  • 選択した列の前に1つまたは複数の列を挿入する機能を追加しました。
  • Thêm chức năng cho phép chèn một hoặc nhiều dòng phía trước dòng đang chọn.
  • Added functionality to insert one or multiple rows before the selected row.
  • 選択した行の前に1つまたは複数の行を挿入する機能を追加しました。
  • Thêm chức năng cho phép người dùng tùy chỉnh màu của ô: đang chọn, đóng băng, nổi bật, ô trống,...
  • Added functionality allowing users to customize cell colors: selected, frozen, highlighted, empty cells, etc.
  • ユーザーがセルの色をカスタマイズできる機能を追加：選択中、固定、ハイライト、空のセルなど。
Phiên bản 2.0.0.1 Version 2.0.0.1 バージョン 2.0.0.1
  • Sửa các lỗi đã được báo cáo.
  • Fixed reported bugs.
  • 報告されたバグを修正しました。
  • Tối ưu bộ xử lý xuất excel giúp cho việc xử lý nhanh và ít tốn ram hơn.
  • Optimized Excel export processor for faster processing and reduced RAM usage.
  • Excel エクスポートプロセッサを最適化し、処理速度の向上と RAM 使用量の削減を実現しました。
  • Nâng cấp lịch sử thao tác (hiện tại có thể hoàn tác tối đa 10000 lần).
  • Enhanced operation history (now supports up to 10,000 undo operations).
  • 操作履歴を強化（最大10,000回の取り消しをサポート）。
  • Thêm hệ thống kiểm tra cập nhật.
  • Added update check system.
  • アップデートチェックシステムを追加しました。
  • Thêm chức năng cho phép xử lý xóa ô của dòng (điều này đáp ứng cho việc xử lý file csv có số cột không đồng nhất).
  • Added functionality to delete cells in a row (supports CSV files with inconsistent column counts).
  • 行内のセル削除機能を追加（列数が不均一な CSV ファイルの処理に対応）。
  • Thêm xử lý cho phép người dùng nhấp đúp vào grid để mở file khi grid đang trống.
  • Added support for double-clicking on the grid to open a file when the grid is empty.
  • グリッドが空のときにダブルクリックしてファイルを開く機能を追加しました。
  • Thêm xử lý cho phép người dùng copy và dán cả file (sẽ thực hiện đọc dữ liệu file giống như đang mở file).
  • Added support for copying and pasting entire files (reads file data as if opening the file).
  • ファイル全体のコピー＆ペースト機能を追加（ファイルを開くようにデータを読み込みます）。
Phiên bản 2.0.0.0 Version 2.0.0.0 バージョン 2.0.0.0
CẬP NHẬT LỚN MAJOR UPDATE メジャーアップデート
  • Nâng cấp từ phiên bản 1.0 với nhiều tính năng mới.
  • Upgraded from version 1.0 with many new features.
  • バージョン1.0から多くの新機能を追加してアップグレードしました。
  • Tối ưu ram (hiệu suất tương tương notepad++ và wps,...) cho phép xử lý file lớn trên 2GB trở lên.
  • Optimized RAM usage (performance comparable to Notepad++ and WPS) enabling processing of large files over 2GB.
  • RAM を最適化（Notepad++ や WPS と同等のパフォーマンス）し、2GB 以上の大きなファイルの処理を可能にしました。
  • Cải tiến hệ thống undo/redo hoàn chỉnh với mọi thao tác với dữ liệu như thêm, sửa, xóa cột/dòng, đóng băng,... (trước đây chỉ có xử lý được với ô dữ liệu).
  • Improved complete undo/redo system for all data operations including adding, editing, deleting columns/rows, freezing, etc. (previously only supported cell data).
  • 完全な元に戻す/やり直しシステムを改善し、列/行の追加、編集、削除、固定など、すべてのデータ操作に対応（以前はセルデータのみ対応）。
  • Cải tiến chế độ copy và dán dữ liệu, giờ đây bạn có thể thao tác copy và dán hàng loạt ô từ excel sang CSVEditor và ngược lại.
  • Enhanced copy and paste mode, now you can copy and paste multiple cells from Excel to CSVEditor and vice versa.
  • コピー＆ペーストモードを改善し、Excel から CSVEditor への複数セルのコピー＆ペーストとその逆が可能になりました。
  • Cải tiến xử lý với cột và hàng được tối ưu hơn, hiện tại có thể đóng băng, ẩn hiện tại bất kỳ vị trí nào.
  • Improved column and row handling with better optimization, now supports freezing and hiding at any position.
  • 列と行の処理を改善し、最適化を向上。任意の位置での固定と非表示をサポートするようになりました。
  • Thêm xử lý lọc dữ liệu theo cột, cho phép kết hợp lọc nhiều cột tương tự excel.
  • Added column-based data filtering, allowing combined filtering of multiple columns similar to Excel.
  • 列ベースのデータフィルタリングを追加し、Excel と同様に複数列の組み合わせフィルタリングが可能になりました。
  • Tối ưu hóa tìm kiếm và thay thế ô dữ liệu cho phép thực hiện nhanh và ít tốn ram hơn.
  • Optimized cell data search and replace for faster execution and reduced RAM usage.
  • セルデータの検索と置換を最適化し、高速実行と RAM 使用量の削減を実現しました。
